# Blog API
A simple RESTful API for creating, reading, updating, and deleting blogs. Includes user authentication with JWT.

## Technologies Used
- Node.js
- Express
- MongoDB (Mongoose)
- JSON Web Token (JWT) for authentication
- bcryptjs for password hashing
